功能说明

单据单号这个功能是在表单和流程中进行使用，这个功能可以在单据上生成递增的单号 ，比如 XSDD-20171205-00002这种编号。

也可以通过API接口调用产生单号。

操作步骤

1.添加单据单号

属性说明:

别名：租户中别名独一无二，不能重复

规则：比如图中 XSDD-{yyyy}{MM}{DD}-{NO} ，这个规则的意思是会产生XSDD-20171205-00002这样的编号。

生成方式：

每天：每天重新生成NO编号 ，比如0001开始增长，第二天又从0001开始增长。

每周：每一周重新生成NO编号

每月：每月重新生成NO编号

每年：每年重新生成NO编号

自动增长: 编号一直累加不中断。

初始值：一开始单号的值，默认从1开始。

步长：流水号每次递增的量，默认为1.


2.通过API接口产生流水号


@Resource
SysSeqIdManager sysSeqIdManager;

String no=sysSeqIdManager.genSequenceNo(“DDH“,“1“);
