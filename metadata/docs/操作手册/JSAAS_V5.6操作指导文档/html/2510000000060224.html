【操作路径】

进入单揣数据列表定义配置界面，如下所示：



以上表单的几点说明：

  1. 启动流程管理：若设置是，则表示在管理列表中对流程展示及进行操作。

  2. 绑定表单方案：表示在列表中可以增加项目信息，展示的表单。

  3. 启用导航树：在列表的左边显示导航树

  4. 数据展示类型：若为数据列表，则数据以数据列表展示，若为树列表，则数据以树列表来展示

  5. 默认的SQL：表示列表进来后的加载的数据列表。

  6. 若数据来自其他库，需要建立其他数据源

  7. 查询的表名，一般为表单的标识键前缀加w_.
